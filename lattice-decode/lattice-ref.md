# Lattice Reference — For Text Analysis Agents

Compact reference for analyzing ancient texts through the lattice.
Everything you need to produce spec files and story files.


## The Equation

  ƒ(τ,χ,μ,φ) = τᵃ·χᵇ·μᶜ·φᵈ

  τ = time       χ = distance     μ = mass       φ = charge
  Even coordinate / 2 = exponent. Odd = force active.

Everything in physics, computing, and EE is these four with exponents.
  voltage = D²M/(T²Q)   bandwidth = M/T   speed = D/T   energy = D²M/T²


## Forces (4 binary choices = 16 bonds)

  π = read       π̄ = write       (projection / antiparticle)
  ∘ = call       ∘̄ = return      (composition / inverse)
  ∮ = advance    ∮̄ = rewind      (contour integral / reverse)
  δ = test       δ̄ = guard       (Kronecker delta / complement)

Overbar = antimatter. Write IS the antimatter of read.
Forward = consume. Backward = produce.

## Shells = Complexity Layers

  shell 1 = linear (speed, density, frequency)
  shell 2 = quadratic (energy, pressure, acceleration)
  shell 3 = cubic (power, intensity, jerk)
  shell 4 = quartic (snap, geodesic)


## Standard Model = Computation

  quarks = logic gates: AND (up), OR (down), NOT (weak force)
  3 gates = 3 color charges. Completeness = confinement.
  proton = ADD (never decays). neutron = SUB (proton + NOT).
  leptons = CPU FLAGS: electron = SF, neutrino = ZF.
  spin ½ = write (exclusive, fermion). spin 1 = read (shared, boson).
  Pauli exclusion = no two writers to same address.


## Consciousness Shells

  π+δ         = perception    (sensor)
  π+∘+δ       = experience    (processing, DAG)
  π+∘+∮+δ     = consciousness (self-referential loop = hylo)

∮ is the key. Without ∮ = feed-forward. With ∮ = output loops back.


## Key Physics Mappings

  dark matter  = written but unread buffer entries (mass, no photons)
  dark energy  = empty ring slots (negative pressure from unfilled capacity)
  observation  = any π (read). No consciousness required.
  classical    = hylo (all 4 forces). quantum = bind (∘∮δ, no π).
  decoherence  = unwanted π. entanglement = non-local shared state.


## Language Mappings

  Parts of speech: noun=μ, verb=τ, adj=φ, adverb=φ+τ, prep=χ, pronoun=μ+φ
  Antonyms = sign flip. Synonyms = distance ≤2. Metaphor = cross-domain match.
  16 bond patterns = 16 linguistic relationship types (see below).

  identity     = self-reference
  π            = reference (pronoun → noun)
  ∘            = invocation (verb calls arguments)
  ∮            = continuation (list, sequence)
  δ            = evaluation (adjective evaluates noun)
  π∘           = traversal (subject → object)
  ∘δ           = conditional (if-then)
  ∘∮δ          = causation (because/therefore)

  raqia = hammered sheet. NOT "firmament."
  shavat = ceased/stopped. NOT "rested."

  tselem = pattern/mold. NOT just "image."

## Method: How To Analyze A Text

1. GET THE SOURCE. Original language, not translations. Translations
   lose structure. Hebrew, Greek, Ge'ez, Chinese — work from roots.

2. IDENTIFY KEY VERBS. What forces are active? Map each verb to its
   bond pattern. "created" might be bara (π̄, write from nothing) or
   asah (π∘, fold from existing). The distinction matters.

3. IDENTIFY KEY NOUNS. What dimension does each noun live on?
   Physical things = μ. Spatial things = χ. Time things = τ.
   Signal/distinction things = φ. Most nouns have a primary dimension.

4. MAP RELATIONSHIPS. Between-verse relationships are bonds.
   Does A reference B? (π). Does A cause B? (∘∮δ). Does A loop
   back to A? (∮). What's the bond graph of the passage?

5. LOOK FOR PATTERNS. Repeated structures = ∮. Paired opposites
   = antimatter. Ascending complexity = shell climbing. Seven-fold
   structures = palms. Twelve-fold = gates×dims.

6. CHECK AGAINST CODEX. Does the formula for a concept match
   where the text places it? If the text puts "light" before "sun,"
   that's φ before τ — signal before clock. Dependency order.

7. CONFIDENCE LEVELS. Linguistic findings (root meanings, grammar)
   = high confidence. Structural parallels (boot sequence, bug report)
   = medium. Specific coordinate assignments = speculative.


## Output Format: Spec File

For each verse or section:

  ### [verse reference]
  Source: [original text]
  Transliteration: [romanized]
  Word-for-word: [literal gloss with | separators]
  Forces: [which of π,∘,∮,δ are active]
  Dimensions: [which of τ,χ,μ,φ are referenced]
  Bond pattern: [the combination, e.g., π∘δ = map]
  Lattice rendering: "[the text restated in lattice terms]"
  Notes: [anything interesting — verb choice, structural parallel, etc.]
  Confidence: [high/medium/low for each claim]


## Output Format: Story File

Write the text as if explaining to someone who understands computing
and physics but has never read the original. Use the lattice vocabulary
naturally — don't force it, but don't avoid it. The story should:

- Lead with what happens, not what it "means"
- Use the original verbs accurately (bara ≠ asah)
- Point out when the text structure reveals something translations hide
- Be direct. No hedging, no "perhaps." State it and mark confidence.
- Let the reader feel the discovery — the moment the pattern clicks

Voice: confident where the evidence is strong, honest where it's
speculative, excited where independent paths converge.
