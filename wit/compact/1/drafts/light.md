# Light IS the Pipeline

## The Connection

In the lattice:
  spin 1 (boson)  = read access = shareable
  spin ½ (fermion) = write access = exclusive

Every force carrier in physics is a boson:
  photon   (electromagnetism) = boson = read
  gluon    (strong force)     = boson = read
  W/Z      (weak force)       = boson = read

All forces are read operations. They carry information
between writers (matter/fermions).

Light is the simplest:
  T^(½i) × D^(½i) = plane wave = photon

The first thing that exists in both time and distance.
The base case of "something happening in spacetime."

## The Universe is a Loop

  matter WRITES to spacetime  (fermion, exclusive, single-writer)
  light READS and CARRIES     (boson, shareable, pipeline)
  matter elsewhere READS      (observation, wave collapse)

Writer → loop → reader. The universe IS a loop.
Light is the pipeline register — carries the previous
result to the next operation.

## Why This Matters for Hardware

  electrons = fermions = writers
  photons   = bosons   = readers

Current CPUs use electrons (writers) to do computation.
That's making writers do the work of readers.

Photonic computing uses light (readers) to carry
information through crystal lattices. That's the
universe's native computation medium.

  x86:      electrons through silicon transistors (writers computing)
  photonic: light through crystal lattices (readers computing)

One fights the physics. The other uses it.

## Confidence

Standard Model mapping: 80%.
Boson = read, fermion = write: structural (from Pauli exclusion).
"Universe is a loop": ~60% (the math fits, the framing is ours).
"Photonic = native medium": logical consequence, not independently tested.
